1方法:(修饰符可以是一个或多个)
    修饰符 返回值类型 方法名(参数){
	return;

    }
    或者:
    修饰符 void 方法名(参数){
    }



方法的使用:
   1.根据要求进行分析,确定返回值类型和参数
   2.函数名称要起一个666的名字,如getSum,getTotal,参数列表的名称也一定起的有意义
   3.return后面不能写任何语句
   4.要注意返回值和所定义的返回值类型相同
   5.最重要的是,作为一个优秀的程序员,要认真的做好每个方法的详细注释



方法的重载:
    指在一个类中,方法名称相同,而参数类型不同,与修饰符和返回值类型无关.叫做方法的重载
    参数列表:数量,类型,顺序不同
    重载方法调用:JVM通过方法的参数列表,调用不同的方法

补充:
   在方法传参数时,有一种特殊的形式要注意,就是可以传任意个参数.
   例:
      public void test(int...a){
          for(int i=0;i<a.length;i++){
          }
      }
    这种相当于其实a就可以当作一个数组来操作,但是在传入的时候只传单个参数即可.
    test(1,2,3);
