正则表达式(RegExp):
  1.用来定义字符串组成的规则.
  2.用来检索某些单词.
规则:
  单个字符:检查用户输入的单个字符是否符合定义的规则.
           [ab]代表单个字符只允许出现a或b.   [a-zA-Z0-9_]:只允许大小写字符和数字,下划线.
           
           说明: 特殊字符:[\d]代表单个数字字符[0-9].      
                          [\w]代表26个字母和数字,下划线[0-9a-zA-Z_]


常用操作符:
       .  :  表示任意单个字符
       ? :  前面一个字符出现0次或1次.	
       * :  前面一个字符出现0次或多次.
       +:  前一个字符出现1次或者多次.
       []:  字符集,对单个字符给出取值范围,       比如[ab]:代表单个字符只能出现a或b.
     [^]:  非字符集,对单个字符给出排除范围,   比如[!ab]:代表单个字符既不能是a也不能是b.
       |  :  左右表达式任意一个
  {m,n}:  扩展前一个字符m-n次.(如果m缺省{,n},代表最多出现n次.如果n缺省{m,},代表最少出现m次)
     {m}:  扩展前一个字符至m次.
       ^ :  匹配字符串开头,^abc-->表示字符串abc且必须在字符串的开头
       $ :  匹配字符串结尾, $abc-->表示字符串abc且必须在字符串的结尾
       () :  分组标记,把字符当作一个整体来看,内部只能使用 | 操作符,(abc)-->表示abc,(abc|def)-->表示abc,def.
      \d :  单个数字,等价于[0-9]
      \w :  单个数字+字母,等价于[a-zA-Z0-9],再加一个下划线.
      
       检测任意字符串:(.*?)
       匹配任意空字符串,包括空格,制表符,换页符等:(\s+)?

       例:[\w]{0,16}:表示只能出现字母,数字,并且位数在0-16之间.
  
  开始结束符号:开始^    结束$(测试时一定要加这两个符号,否则无法正确定义规则)


创建方式:
   1.var reg=new RegExp("正则表达式");  //加""
     说明:此方式输入\w时前面要加转义符\.否则不能正确识别\w.
     例:
        var str=new RegExp("^\\w{0,10}\\d$"); //加了开始,结束符号,转义符.
        var result=str.test("fhjg367");
        alert(result);
   
   2.var reg=/正则表达式/;              //不加""
     例:
        var str=/^\d{0,11}$/;
        var result=str.test("11234567890677");
        alert(result);

手机号的正则表达式:
   str=/^1[34578]\d{9}$/
邮箱正则表达式
   ^[a-zA-Z0-9_-]+@[a-zA-Z0-9_-]+(\.[a-zA-Z0-9_-]+)+$
汉字的正则表达式
   [\u4E00-\u9FA5]
